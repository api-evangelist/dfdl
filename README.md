# DFDL (dfdl)
Data Format Description Language (DFDL) is an open standard for describing data formats used in binary and text files and data streams. DFDL is used to describe the format of existing data so that it can be parsed or unparsed (generated) using a DFDL processor, enabling integration with legacy and modern data systems.
